# 定价策略

> 最后更新：2026-07-23
>
> 本文档记录纪念券定制业务的成本模型与定价结论。讨论过程中的作废假设（如 100 张同图批量价、家用自印路线）不收录，只保留当前有效的口径。

## 商业模式前提

- 激活码模式：一码 = 一次纪念券设计 + 一张实体券制作 + 邮寄，设计过程可任意使用生图 / LLM，不再额外收费。
- 产品是定制品：每个顾客的图都不同，每一单对打印店都是「多款少量」，**永远按打样价付费，享受不到批量价**。
- 当前阶段全部外发打印店制作，不考虑家用自印。

## 单张成本模型

### 制作成本（¥23–26）

| 成本项 | 金额 | 说明 |
|---|---|---|
| 数码彩印（含 120g 棉纸） | ¥3.5–5 | 外发打印店打样价（实测：4 款各 5 张共 20 张 ¥70）；定制场景常态价，详见 [打印店沟通记录](../02-印刷制作/打印店沟通记录.md) |
| 丝印 + 荧光油墨 | ¥1–11 | **按调墨批次算，不按张算**，见下方专项分析 |
| AI 生图 | ¥2–2.5 | ~¥0.5/次 × 4–5 次迭代（生成耗时 1–2 分钟/次，用户耐心天然限制迭代上限） |
| LLM 对话 | ~¥1 | 计划用 DeepSeek，成本很低 |
| 废品损耗 | ¥1.5–3 | 废品率 5–10% × 已投入成本 |

### 履约成本（¥15）

| 成本项 | 金额 | 说明 |
|---|---|---|
| 快递 | ~¥10 | 早期无快递合作，散单价格 |
| 包装 | ~¥5 | 分解见下 |

包装 ¥5 的分解：

- **¥2 保命**：纸币保护袋 ¥0.2–0.5 + PVC 硬卡套 ¥0.5–1（120g 棉纸软，无硬卡套必折）+ 气泡信封 ¥0.5–1
- **¥2 塑魂**：定制封套/展示卡 ¥1–2 + 说明卡 ¥0.3–0.5（写清「365nm 紫外灯照射可见隐藏纹样」——产品最大记忆点，不告知就白做）
- **¥1 机动**

### 油墨成本专项（关键认知）

材料单价：软凝胶 ¥117/230ml ≈ ¥0.5/g；变色粉 ¥15/15g = ¥1/g。调一杯 20g 标准墨（粉 10%）≈ ¥11。

- 单张券实际消耗墨仅 0.1–0.2g（面值镂空面积 ~2.5cm² × 湿墨层 ~25μm + 网面残墨），**一杯 20g 墨物理上可印 100+ 张**。
- 墨的瓶颈从来不是墨量，而是寿命（丙烯体系密封后 1–2 天）。零星单时代印 3 张就废掉一杯，损耗率 97%。
- 因此 ¥11 本质是「开机费」，对策有两条：
  1. **按当日单量定杯量**：0.1g 精度秤调 5g 无压力。零星单调 5g（¥2.75），损耗压到 ¥2 左右；单多再调 10–20g。
  2. **凑批生产**：发货承诺写「下单后 2–3 天内制作发出」，攒单凑一批一起调墨。墨的寿命 1–2 天与发货窗口天然匹配。
- 待验证：安全线淡墨（粉 5%）可用面值浓墨的剩墨掺凝胶稀释获得，一次调墨覆盖两道工序，不必单独调第二杯。

## 总成本与毛利

| 状态 | 单张总成本 | ¥49 定价毛利 |
|---|---|---|
| 零星单（一单调 20g 墨） | ~¥34 | ~30% |
| 零星单（按 5g 小杯调墨） | ~¥26 | ~47% |
| 凑批 5 张一批墨 | ~¥25 | ~50% |

结论：¥49 定价下毛利率 30–50%，波动几乎全来自油墨调杯量和凑批纪律——这是排产问题，不是成本结构问题。

## 定价结论

**定价公式：max（可变成本 × 2.5, 市场对标区间）× 阶段策略。**

- **硬底线 ¥30**：低于此价长期必亏。
- **主推价 ¥49**：稳态毛利 ~50%。
- **¥39**：首张体验价或 2 张以上单价，毛利 ~35%，不做默认价。
- **放弃 ¥29.9 引流价**：全外发成本结构下是纯贴钱，且贴的钱给了打印店和快递，顾客无感知。早期换种子内容用「¥39 首单 + 晒图返 ¥10」之类方式，返现能换回社区内容。
- **多张阶梯价**：1 张 ¥49 / 2 张 ¥79 / 3 张 ¥99（同图加印边际成本仅 ¥15–20，且同批生产摊薄油墨和快递）。成本结构里快递+包装 ¥15、彩印 ¥5 都随张数摊薄，**提客单比压成本有效得多**，这是定价策略的核心。
- **面值不影响定价**：换面值只是换一块网版，边际成本为零；但面值种类数影响制版固定成本，首期限定 3–4 种面值。
- **AI 成本不做限制**：生图 ¥0.5/次 + 生成耗时天然限速，¥49 定价完全罩得住任意迭代。

### 定价的价值依据（为什么不按成本加成定低价）

- 千人千样 + 独立编号 = 收藏品逻辑，不是印刷品逻辑；印刷品按纸墨计价，收藏品按稀缺性计价。
- 多层工艺叠加（丝印触感面值 + UV 荧光隐藏纹样）提供「把玩时刻」，是情绪价值和传播点。
- 市场对标：定制色纸 ¥15–30、定制吧唧/立牌 ¥20–40、同人收藏品 ¥30–100+；本产品工艺复杂度高于色纸吧唧，定中上区间合理。

### 包邮策略

快递成本直接打进码价，不做「码价 + 邮费」两段式——电商场景包邮转化率明显更好。
